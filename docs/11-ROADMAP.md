# CalorieTracker - Roadmap

**Implementado (v0.1.0):**
- [x] MVP: Dashboard, diario, exercicios, perfil, peso
- [x] Onboarding completo (5 steps)
- [x] Sistema Noom Color (green/yellow/orange)
- [x] Calculo BMR (Mifflin-St Jeor + Katch-McArdle)
- [x] Projecao de peso (12 semanas)
- [x] Montador de dieta automatico (macro-first)
- [x] Base Alimentar (alimentos fixos)
- [x] Combos/Receitas personalizadas
- [x] Deteccao de alimentos similares
- [x] Streak de dias consecutivos
- [x] Deploy Vercel + Neon PostgreSQL

**Planejado:**
- [ ] Autenticacao multi-usuario (NextAuth.js ou Clerk)
- [ ] PWA com notificacoes push
- [ ] Scan de codigo de barras (camera)
- [ ] Integracao com APIs de alimentos (TACO, FatSecret)
- [ ] Historico de refeicoes com graficos semanais/mensais
- [ ] Exportar dados (CSV/PDF)
- [ ] Compartilhar dieta com nutricionista
- [ ] Integracao com wearables (Google Fit, Apple Health)
- [ ] Modo escuro automatico
- [ ] Receitas da comunidade
